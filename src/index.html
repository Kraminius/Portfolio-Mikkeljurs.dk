Interactive Database Navigator: Seamless SQL Exploration & Table Management with JavaFX

"Interactive Database Navigator" is a GUI-driven SQL table management tool developed in JavaFX. It provides an intuitive way to view, manage, and manipulate database tables without writing SQL queries. Users can explore table structure and content, execute custom SQL queries, perform CRUD operations on rows, import/export data in CSV format, and manage database connection profiles. Multithreading ensures smooth user experience, even for complex tasks. The project was unit-tested using JUnit and utilized Git for version control, following Agile development practices.
